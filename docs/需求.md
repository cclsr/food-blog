# 美食博客 · 需求文档

> 一个公开的个人美食博客，包含三大模块：美食分享、朋友菜单、照片墙。
> 本文档是开发的「说明书」，随做随改。

## 一、定位与使用者

- **性质**：公开站点，任何访客无需登录即可浏览所有内容。
- **发布者**：站长（本人）登录后台后发布、编辑、删除内容。
- **访客**：只读浏览（评论功能为可选项，二期再加）。

## 二、技术栈

| 关注点 | 选型 |
|---|---|
| 框架 | Next.js（App Router，前后端一体） |
| 样式 | Tailwind CSS + shadcn/ui |
| 数据库/存储/登录 | Supabase（Postgres + Storage + Auth） |
| 部署 | Vercel（连 GitHub 自动发布） |

## 三、功能模块

### 1. 美食分享（核心）
- 列表页：卡片网格，显示封面图、标题、简介、日期、标签。
- 详情页：大图、正文（支持图文）、可选的菜谱步骤区。
- 字段：标题、封面图、正文、标签、（可选）食材清单、（可选）步骤。
- 标签筛选（如「家常菜」「烘焙」「探店」）。

### 2. 朋友菜单
- 一份或多份共享菜单清单（如「周末聚餐想吃的」「想打卡的餐厅」）。
- 每个条目：菜名/店名、备注、是否已完成（打卡）。
- 一期由站长维护展示；二期可考虑开放朋友协作编辑。

### 3. 照片墙
- 瀑布流/网格图片展示。
- 图片可选关联到某篇美食帖（点击跳转）。
- 支持简单说明文字（caption）。

## 四、页面清单（一期）

- `/` 首页：精选美食 + 最新照片 + 入口导航
- `/food` 美食列表 ·  `/food/[slug]` 美食详情
- `/menu` 朋友菜单
- `/gallery` 照片墙
- `/admin` 站长后台（发布/编辑，二期接 Auth）
